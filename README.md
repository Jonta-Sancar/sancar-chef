# EXERCÍCIO DE LÓGICA

## Contexto: concurso de culinária

## Momento: Seleção de competidores

## Regra para classificação - critérios iguais ou superiores a:

| Qualidade dos pratos | Tempo de entrega | Apresentação |
| 6.5 | 15% | 7 |

## método de avaliação:
  Os competidores farão 3 pratos, a média das pontuações alcançadas devem ser iguais ou superiores aos critérios de classificação.

## Valores fixos:
- Pratos a serem feitos (3 pratos)
- Duração máxima para cada prato (tempo em minutos)
- média de cada critério

## Valores capturados do usuário (via terminal):
- Tempo de realização de cada prova
- pontuação de qualidade de cada prova
- pontuação de apresentação de cada prova
- nome do competidor
- idade do competidor
- especialidade do competidor


## entrega final - mensagem ao usuário, deve conter:
- se foi aprovado ou não para a próxima etapa
- nome, idade e especialidade do competidor
- uma mensagem personalizada para cada cenário (aprovação ou reprovação)

# Etapa 2

## Para fazer a repescagem
* Perder em no máximo 2 critérios
* A diferença de nota não pode superar 20%

## para os que passaram eles se dividem em categorias
* na média       == PASSADOR
* acima da média == TIRA ONDA
* fechou tudo    == CHEFE

## Essas novas regras precisam corresponder a variações na mensagem final.
